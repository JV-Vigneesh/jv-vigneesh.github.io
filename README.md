## 📁 Repository Structure
This repository contains my personal portfolio website, designed and built from scratch with standard web technologies, responsive styling, and dynamic components:

*   `index.html`] — The main dashboard, featuring an immersive home hero banner, a resume timeline, projects grid, freelance capabilities showcase, and a functional contact form.
*   `projects.html`] — A dedicated library showcasing all built projects with interactive category filters.
*   `project-detail.html` — A dynamic detail page displaying rich metadata, role descriptions, features, and screenshots for a chosen project.
*   `projects-data.js` — The central project database holding structured data (titles, descriptions, tech stacks, and feature lists) rendered dynamically by JS.
*   `script.js` — Handles interactivity, interactive filtering, header scroll effects, and custom mouse-movement 3D card tilt transformations.
*   `style.css` — Custom stylesheet showcasing premium glassmorphism, tailored HSL color palettes, custom typography (Space Grotesk, Inter, JetBrains Mono), responsive design rules, and micro-animations.
